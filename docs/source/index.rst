OBS Script Software
===================

Overview
--------

This program allows for simpler creation of the OBS files required for the NANTEN2 telescope through:

* FITS image viewer
* Compatability with rectangular pyregion REG files
* Interactive map specification
* Automated conversion between coordinate systems
* Automated computation for codependent OBS parameters

--------
